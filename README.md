

# 🚗【手车互联】：既然带了手机，为什么还要带车钥匙？

你是否也经历过以下场景：
1. 走到车边，像特工一样掏出手机，打开那个加载半天的车控 App，只为开个锁？
2. 停好车走远了，突然一阵心慌：“我刚才锁车了吗？”然后顶着大太阳折返回去拉一下车门？
3. 官方的蓝牙钥匙由于各种神秘原因（手机休眠、权限被杀、心情不好）偶尔罢工？

**作为一个能用代码解决就绝不自己动手的懒人，我写了这个 Tasker 方案。**

---

### 🛠️ 它能干啥？（翻译：我是怎么偷懒的）

*   **“蓝牙跟踪狂”模式 (RSSI 定位)**：
    它会盯着你手机和车的蓝牙信号强度。离得近了（比如刚到车门边），它就自动开锁；离得远了，它就自动锁车。不用掏手机，不用点屏幕，主打一个“目中无人”。
    
*   **防止“误伤”的脑子**：
    它不是简单的“断开就锁”。锁车前它会先问问车：“你熄火了吗？门关严了吗？” 如果你还在车里坐着发呆，它绝不会把你反锁在里面。

*   **爱操心的微信提醒**：
    自动锁车成功后，它会给你发个微信：“主子放心，车锁好了。”万一门没关紧锁不上，它也会立刻报警：“别走了！门还缝着呢！”

*   **极客的“虚荣”面板**：
    顺手做了一个 UI 面板。电量、续航、胎压一目了然。虽然官方 App 也有，但这个是我自己搓的，看着爽，而且快。

*   **专治各种不服 (VPN 自动关)**：
    如果你手机常年挂着 VPN 导致控车接口报错，这个脚本还会自行动手，尝试重置网络环境。逻辑就是：只要我不尴尬，尴尬的就是 Bug。

---

### 🔧 怎么调教它？

这东西不是“开箱即用”，它需要你有一点点折腾精神：
1.  **导入方案**：把那个 XML 喂给 Tasker。
2.  **人肉校准**：站在车门边看看信号是多少（比如 -75），走开五步看看是多少（比如 -85）。把这两个数填进设置界面。
3.  **给它权限**：别让安卓系统把它给杀了，否则它就成了“人工智障”。

### ⚠️ 一些大实话（免责声明）

*   **RSSI 是玄学**：蓝牙信号会受人体遮挡、手机放兜里的姿势影响。如果偶尔没感应到，请保持优雅，手动点一下。
*   **别把钥匙真扔了**：虽然它很智能，但万一手机没电了，或者服务器抽风了，你还是需要物理钥匙救命的。
*   **严禁商用**：这就是个个人折腾的小玩具，欢迎车友交流，想拿去卖钱的请绕道。
